# SPAjs

This is the vs code snippets extension for  [SPAjs](https://spa.js.org).

![Overview](https://media.giphy.com/media/3ohs4fuOhWNBI991Ha/giphy.gif)

## Snippets
Shortcuts In **[.html]** file:

spa$

---
In **[.js]** file:

spa$

spa$extend

spa$render

spa$..

---
