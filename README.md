# SPAjs

This is the vs code snippets extension for  [SPAjs](https://spa.js.org).

![Overview](images/overview.gif)

## Snippets
Shortcuts In **[.html]** file:

spa$

---
In **[.js]** file:

spa\$
spa\$extend
spa\$render
spa\$..

---
